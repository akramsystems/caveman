**Claude Code skill. Make Claude talk like caveman.**

## Install

Symlink the repo into your personal skills directory:

```bash
ln -s "$PWD" ~/.claude/skills/caveman
```

## Use

In any Claude Code session:

```
/caveman
```

Or just say *"caveman mode"*. Say *"stop caveman"* to turn off.
